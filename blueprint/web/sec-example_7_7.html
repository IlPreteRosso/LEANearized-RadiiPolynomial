2.2 Example 7.7: Square Root via Power Series

This section applies the radii polynomial method to prove existence of an analytic branch of \(\sqrt{\lambda }\) near a base point \(\lambda _0 {\gt} 0\). We work in the weighted sequence space \(\ell ^1_\nu \) and verify all bounds to establish existence of a fixed point, then interpret this fixed point as a convergent power series.

\begin{equation*} \| a\| _{\ell ^1_\nu } := \sum _{n=0}^\infty |a_n| \nu ^n {\lt} \infty . \end{equation*}

2.2.2 Cauchy Product and Banach Algebra Structure

Definition 2.2.4 Cauchy product
✓
#

Lean declarations

  • CauchyProduct

The Cauchy product of sequences \(a, b : \mathbb {N} \rightarrow \mathbb {R}\) is

\begin{equation*} (a \star b)_n := \sum _{k=0}^n a_k b_{n-k}. \end{equation*}
Theorem 2.2.5 Closure under Cauchy product
✓
#

Uses

  • Definition 2.2.2
  • Definition 2.2.4

Lean declarations

  • l1Weighted.mem

If \(a, b \in \ell ^1_\nu \), then \(a \star b \in \ell ^1_\nu \).

Proof ▶

By the triangle inequality and rearrangement:

\begin{align*} \sum _{n=0}^\infty |(a \star b)_n| \nu ^n & \leq \sum _{n=0}^\infty \sum _{k=0}^n |a_k| |b_{n-k}| \nu ^n \\ & = \sum _{n=0}^\infty \sum _{k=0}^n |a_k| \nu ^k \cdot |b_{n-k}| \nu ^{n-k} \\ & = \| a\| _{\ell ^1_\nu } \cdot \| b\| _{\ell ^1_\nu } {\lt} \infty . \end{align*}
Theorem 2.2.6 Submultiplicative norm
✓
#

Uses

  • Definition 2.2.2
  • Definition 2.2.4

Lean declarations

  • l1Weighted.norm_mul_le

For \(a, b \in \ell ^1_\nu \):

\begin{equation*} \| a \star b\| _{\ell ^1_\nu } \leq \| a\| _{\ell ^1_\nu } \cdot \| b\| _{\ell ^1_\nu }. \end{equation*}

This makes \((\ell ^1_\nu , \star )\) a Banach algebra.

2.2.3 The Fixed-Point Problem

We seek an analytic function \(x(\lambda ) = \sum _{n=0}^\infty a_n (\lambda - \lambda _0)^n\) satisfying \(x(\lambda )^2 = \lambda \). In coefficient space, this becomes: find \(a \in \ell ^1_\nu \) such that \((a \star a)_n = c_n\) where \(c\) is the parameter sequence.

Definition 2.2.7 Parameter sequence
✓
#

Lean declarations

  • Example_7_7.paramSeq

For \(\lambda _0 \in \mathbb {R}\), the parameter sequence is

\begin{equation*} c_n = \begin{cases} \lambda _0 & n = 0 \\ 1 & n = 1 \\ 0 & n \geq 2 \end{cases}\end{equation*}

This encodes the polynomial \(\lambda _0 + z\) (i.e., \(\lambda \) when \(z = \lambda - \lambda _0\)).

Definition 2.2.8 The map \(F\)

\begin{equation*} F(a)_n = (a \star a)_n - c_n. \end{equation*}

A zero of \(F\) corresponds to a power series squaring to \(\lambda _0 + z\).

2.2.5 Bound Definitions

Definition 2.2.13 \(Y_0\) bound
✓
#

Uses

  • Definition 2.2.8
  • Definition 2.2.11

Lean declarations

  • Example_7_7.Y₀_bound

The \(Y_0\) bound measures the initial defect:

\begin{equation*} Y_0 = \| A \cdot F(\bar{a})\| _{\ell ^1_\nu } \end{equation*}

where \(A\) is the approximate inverse. For diagonal \(A\) with entries \(A_{nn} = 1/(2\bar{a}_0)\):

\begin{equation*} Y_0 = \sum _{n=0}^N |A_{nn} \cdot F(\bar{a})_n| \nu ^n + Y_0^{\text{tail}} \end{equation*}

where \(Y_0^{\text{tail}}\) accounts for the tail contribution.

Definition 2.2.14 \(Z_0\) bound
✓
#

Uses

  • Definition 2.2.11
  • Theorem 2.2.10

Lean declarations

  • Example_7_7.Z₀_bound

The \(Z_0\) bound measures deviation from identity:

\begin{equation*} Z_0 = \| I - A \circ D_{\bar{a}}F\| . \end{equation*}

Since \(D_{\bar{a}}F(h) = 2(\bar{a} \star h)\) and \(A\) is diagonal with \(A_{nn} = 1/(2\bar{a}_0)\):

\begin{equation*} Z_0 = \sup _{n \geq 0} \left|1 - \frac{2(\bar{a} \star e_n)_n}{2\bar{a}_0}\right| \cdot \nu ^n \end{equation*}

where \(e_n\) is the \(n\)-th standard basis sequence.

Definition 2.2.15 \(Z_1\) bound
✓
#

Uses

  • Definition 2.2.11
  • Theorem 2.2.10

Lean declarations

  • Example_7_7.Z₁_bound

In the simple case where \(A^\dagger = D_{\bar{a}}F\), we have \(Z_1 = 0\).

Definition 2.2.16 \(Z_2\) bound
✓
#

Uses

  • Theorem 2.2.10

Lean declarations

  • Example_7_7.Z₂_bound

The \(Z_2\) bound measures Lipschitz continuity of the derivative:

\begin{equation*} \| A \circ (D_c F - D_{\bar{a}} F)\| \leq Z_2 \cdot r \end{equation*}

for all \(c \in \overline{B}_r(\bar{a})\). Since \(D_c F - D_{\bar{a}} F = 2((c - \bar{a}) \star \cdot )\):

\begin{equation*} Z_2 = \frac{2 \| A\| }{\text{(appropriate norm)}}. \end{equation*}

2.2.8 Analytic Interpretation

Having established existence in coefficient space, we now interpret the result as convergence of a power series. We separate this into two levels:

  1. Formal level: Embed \(\ell ^1_\nu \) into the ring of formal power series \(\mathbb {R}[\! [X]\! ]\)

  2. Analytic level: Evaluate the series at points in the disk of convergence

2.2.9 Branch Selection

The equation \(y^2 = \lambda \) has two solutions: \(\pm \sqrt{\lambda }\). We show that the power series selects the positive branch.

Theorem 2.2.32 Positive branch identification
✓
#

Uses

  • Corollary 2.2.30
  • Lemma 2.2.31
  • Definition 2.2.12

Lean declarations

  • Example_7_7.analyticSolution_eq_sqrt

Suppose:

  1. \(\lambda _0 {\gt} 0\)

  2. \(\lambda {\gt} 0\)

  3. \(|\lambda - \lambda _0| \leq \nu \)

  4. \(\tilde{a}_0 {\gt} 0\)

Then \(\texttt{eval}(\tilde{a}, \lambda - \lambda _0) = \sqrt{\lambda }\).

Proof ▶

Step 1: Evaluate at \(z = 0\): \(\texttt{eval}(\tilde{a}, 0) = \tilde{a}_0\).

Step 2: From \(\texttt{eval}(\tilde{a}, 0)^2 = \lambda _0\) and \(\tilde{a}_0 {\gt} 0\): \(\tilde{a}_0 = \sqrt{\lambda _0} {\gt} 0\).

Step 3: Since \(\texttt{eval}(\tilde{a}, \lambda - \lambda _0)^2 = \lambda {\gt} 0\), we have \(\texttt{eval}(\tilde{a}, \lambda - \lambda _0) \neq 0\).

Step 4 (Positivity by IVT): Suppose for contradiction that \(\texttt{eval}(\tilde{a}, \lambda - \lambda _0) {\lt} 0\). Since \(\texttt{eval}(\tilde{a}, 0) {\gt} 0\) and \(z \mapsto \texttt{eval}(\tilde{a}, z)\) is continuous (uniform convergence of power series), by the Intermediate Value Theorem there exists \(c\) between \(0\) and \(\lambda - \lambda _0\) with \(\texttt{eval}(\tilde{a}, c) = 0\).

But \(\texttt{eval}(\tilde{a}, c)^2 = \lambda _0 + c\), so \(0 = \lambda _0 + c\), giving \(c = -\lambda _0 {\lt} 0\).

Case 1: If \(\lambda \geq \lambda _0\), then \(c \in [0, \lambda - \lambda _0]\) requires \(c \geq 0\), contradicting \(c = -\lambda _0 {\lt} 0\).

Case 2: If \(\lambda {\lt} \lambda _0\), then \(c \in [\lambda - \lambda _0, 0]\) requires \(c \geq \lambda - \lambda _0\), i.e., \(-\lambda _0 \geq \lambda - \lambda _0\), so \(\lambda \leq 0\), contradicting \(\lambda {\gt} 0\).

Therefore \(\texttt{eval}(\tilde{a}, \lambda - \lambda _0) {\gt} 0\).

Step 5: Since \(\texttt{eval}(\tilde{a}, \lambda - \lambda _0) {\gt} 0\) and \(\texttt{eval}(\tilde{a}, \lambda - \lambda _0)^2 = \lambda \), we have \(\texttt{eval}(\tilde{a}, \lambda - \lambda _0) = \sqrt{\lambda }\).

2.2.10 Summary

Theorem 2.2.33 Main result for Example 7.7
✓
#

Uses

  • Theorem 2.2.21
  • Theorem 2.2.32

Lean declarations

  • Example_7_7_Final.example_7_7_main_theorem_rat

For \(\lambda _0 = 1\) and \(\nu = 1/2\), there exists a unique power series

\begin{equation*} \tilde{x}(\lambda ) = \sum _{n=0}^\infty \tilde{a}_n (\lambda - 1)^n \end{equation*}

convergent for \(|\lambda - 1| \leq 1/2\) (i.e., \(\lambda \in [1/2, 3/2]\)) such that

\begin{equation*} \tilde{x}(\lambda ) = \sqrt{\lambda }. \end{equation*}

The coefficients \(\tilde{a}\) are within distance \(r_0\) of the approximate Taylor coefficients \((1, 1/2, -1/8, 0, 0, \ldots )\) in the \(\ell ^1_\nu \) norm.

This completes the formalization of Example 7.7, demonstrating:

  1. Application of radii polynomial method to infinite-dimensional Banach algebra

  2. Coefficient-space verification via rational arithmetic

  3. Formal level: Embedding \(\ell ^1_\nu \hookrightarrow \mathbb {R}[\! [X]\! ]\) where multiplication is the Cauchy product by definition

  4. Analytic level: Mertens’ theorem shows evaluation commutes with multiplication

  5. Branch selection via continuity and the Intermediate Value Theorem
