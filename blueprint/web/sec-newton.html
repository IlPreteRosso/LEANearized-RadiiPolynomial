3 Newton’s Method

Newton’s method transforms the problem of finding zeros into the problem of finding fixed points. This section establishes the fundamental equivalence and introduces Newton-like operators.

Definition 3.1 Newton-like map

Let

\(\)E, F\( be Banach spaces, \)f: E F\( a function, and \)A: F E\( a continuous linear map. The \emph{Newton-like map} is defined by \[ T(x) = x - A(f(x)). \] \)

Let \(\)f: E F\( and \)A: F E\( be an injective linear map. Let \)T(x) = x - A(f(x))\( be the Newton-like operator. Then: \[ T(x) = x \quad \iff \quad f(x) = 0. \] \)

Proof ▶

First direction (

\(\)T(x) = x ⇒f(x) = 0\(): If \)T(x) = x\(, then \)x - A(f(x)) = x\(, which gives \)A(f(x)) = 0\(. Since \)A\( is linear, \)A(0) = 0\(. By injectivity of \)A\(, we have \)A(f(x)) = A(0)\( implies \)f(x) = 0\(. Second direction ( \)f(x) = 0 ⇒T(x) = x\(): If \)f(x) = 0\(, then \)T(x) = x - A(f(x)) = x - A(0) = x - 0 = x\(. \)

Definition 3.4 Classical Newton operator
#

If

\(\)f C^2(U, R^n)\( where \)U ⊂R^n\( is an open set, the \emph{classical Newton operator} is given by \[ T(x) := x - (\text{Df}(x))^{-1} f(x). \] \)
