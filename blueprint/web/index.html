0.1 Radii polynomials in finite dimensions

Definition 0.1.1 Finite-dimensional data
#

Lean declarations

  • RP.Data

Fix a norm \(\| \cdot \| \) on \(\mathbb R^n\) (or \(\mathbb C^n\)). Let \(f:\mathbb R^n\to \mathbb R^n\) be \(C^1\), \(\bar x\in \mathbb R^n\) a numerical approximation to a zero of \(f\), and \(A\in M_n(\mathbb R)\) (typically \(A\approx Df(\bar x)^{-1}\)).

Definition 0.1.2 Newton-like map
#

Uses

  • Definition 0.1.1

Lean declarations

  • RP.T

Define \(T:\mathbb R^n\to \mathbb R^n\) by \(T(x)=x-A f(x)\).

Definition 0.1.3 A priori bounds \(Y_0,Z_0,Z_2\)
#

Uses

  • Definition 0.1.1
  • Definition 0.1.2

Lean declarations

  • RP.Bounds

Choose nonnegative \(Y_0,Z_0\) and a function \(Z_2:(0,\infty )\to [0,\infty )\) such that for all \(r{\gt}0\) and all \(c\in B_r(\bar x)\),

\[ \| A f(\bar x)\| \le Y_0,\qquad \| I-ADf(\bar x)\| \le Z_0,\qquad \| A(Df(c)-Df(\bar x))\| \le Z_2(r)\, r. \]
Definition 0.1.4 Radii polynomial
#

Uses

  • Definition 0.1.3

Lean declarations

  • RP.RadiiPolynomial

For \(r{\gt}0\), set

\[ p(r):=Z_2(r)\, r^2-(1-Z_0)\, r+Y_0. \]
Lemma 0.1.5 Derivative bound
#

Uses

  • Definition 0.1.2
  • Definition 0.1.3

Lean declarations

  • RP.DT_bound

For every \(c\in B_r(\bar x)\),

\[ DT(c)=I-ADf(c),\qquad \| DT(c)\| \le Z_0+Z_2(r)\, r =: Z(r). \]
Proof ▶

Immediate from \(DT=I-ADf\) and the \(Z_0,Z_2\) hypotheses.

Lemma 0.1.6 Ball invariance and contraction criterion
#

Uses

  • Definition 0.1.2
  • Lemma 0.1.5

Lean declarations

  • RP.FixedPointRadii

If \(r{\gt}0\) satisfies \((Z(r)-1)r+Y_0{\lt}0\), then \(T(B_r(\bar x))\subseteq B_r(\bar x)\) and \(T\) is a contraction on \(B_r(\bar x)\) with Lipschitz constant \(\le Z(r){\lt}1\).

Proof ▶

Use the mean value inequality for \(T\) together with \(\| T(\bar x)-\bar x\| \le Y_0\) and \(\sup _{c\in B_r}\| DT(c)\| \le Z(r)\).

Theorem 0.1.7 Radii polynomials in finite dimensions
#

Uses

  • Definition 0.1.4
  • Lemma 0.1.5
  • Lemma 0.1.6

Lean declarations

  • RP.RadiiPolynomialFiniteDim

If there exists \(r_0{\gt}0\) with \(p(r_0){\lt}0\), then:

Proof ▶

From \(p(r_0){\lt}0\) we get \(Z(r_0){\lt}1\) and \((Z(r_0)-1)r_0+Y_0{\lt}0\). Apply Lemma 0.1.6 with \(r=r_0\) to obtain (i) and (ii) by the contraction mapping theorem. Then \(T(\tilde x)=\tilde x\) implies \(A f(\tilde x)=0\), hence \(f(\tilde x)=0\) since \(A\) is injective. Finally \(\| I-ADf(\tilde x)\| {\lt}1\) implies \(Df(\tilde x)\) invertible (Neumann series).
