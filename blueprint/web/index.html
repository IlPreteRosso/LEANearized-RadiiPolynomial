RadiiPolynomial

IlPreteRosso

An attempt to formalize the finite dimensional radii-polynomial theorem in Lean. The formalization is based on the book Ordinary Diﬀerential Equations: A Constructive Approach by J.B. van den Berg, M. Gameiro, J.-P. Lessard, J.D. Mireles James, and K. Mischaikow


Notations

Given \(x, y \in \mathbb {R}^{n}\),

\[ x \ll y \text{ if and only if } x_{k} \leq y_{k} \text{ for all } k=1, \ldots , n \]

The default norm on \(\mathbb {R}^{n}\) is the 2 -norm:

\[ \| x\| \stackrel{\text{ def }}{=} \sqrt{\sum _{k=1}^{n}\left|x_{k}\right|^{2}} . \]

Let \(\| \cdot \| _{X}\) denote a norm on a space \(X\). Then

\[ \overline{B_{\epsilon }\left(x_{0}\right)} \stackrel{\text{ def }}{=}\left\{ x \in X \mid \left\| x-x_{0}\right\| _{X} \leq \epsilon \right\} \]

Let \(J\) be an interval and \(\alpha : J \rightarrow X\) where \(X\) is a normed space with norm \(\| \cdot \| _{X}\). The \(C^{0}\) norm of \(\alpha \) is given by

\[ \| \alpha \| _{C^{0}(J)} \stackrel{\text{ def }}{=} \sup \left\{ \| \alpha (t)\| _{X} \mid t \in J\right\} . \]

Let \(M_{n}(\mathbb {R})\) the set a \(n \times n\) matrices with real coefficients and \(M_{n}(\mathbb {C})\) the set a \(n \times n\) matrices with complex coefficients.

Given a matrix \(A=\left\{ a_{i, j}\right\} _{i, j}\) and a vector \(v=\left\{ v_{i}\right\} _{i}\) (real or complex valued), then the elementwise absolute value is represented by \(|A|=\left\{ \left|a_{i, j}\right|\right\} _{i, j}\) and \(|v|=\left\{ \left|v_{i}\right|\right\} _{i}\), where \(|\cdot |\) denotes the absolute value.

The sup norm on \(\mathbb {R}^{n}\) is defined as follows. Given \(x=\left(x_{1}, \ldots , x_{n}\right) \in \mathbb {R}^{n}\)

\[ \| x\| _{\infty } \stackrel{\text{ def }}{=} \max _{k=1, \ldots , n}\left\{ \left|x_{k}\right|\right\} . \]

Given a matrix \(A\), the matrix norm induced by \(\| \cdot \| _{\infty }\) is defined to be

\[ \| A\| _{\infty }=\max _{\| x\| _{\infty } \leq 1}\| A x\| _{\infty } . \]

If \(T: X \rightarrow X\), then its \(n\)-th iterate \(T^{n}\) is recursively defined by \(T^{n}(x)=T\left(T^{n-1}(x)\right)\).

0.1.1 Contraction mapping theorem

Definition 0.1.1 Metric space
#

Lean declarations

  • RP.MetricSpace

A metric space is a pair \((X,d)\) where \(X\) is a set and \(d:X\times X\to [0,\infty )\) satisfies: (i) \(d(x,y)=0\iff x=y\); (ii) \(d(x,y)=d(y,x)\); (iii) \(d(x,z)\le d(x,y)+d(y,z)\).

Definition 0.1.2 Complete metric space
#

Lean declarations

  • RP.Complete

A metric space \((X,d)\) is complete if every Cauchy sequence \((x_n)\) in \(X\) converges to a limit \(x\in X\).

Definition 0.1.3 Lipschitz map and contraction
#

Lean declarations

  • RP.Contraction

Let \((X,d)\) be a metric space. A map \(T:X\to X\) is Lipschitz with constant \(L\ge 0\) if \(d(Tx,Ty)\le L\, d(x,y)\) for all \(x,y\in X\). It is a contraction if it is Lipschitz with some \(L{\lt}1\).

Definition 0.1.4 Picard iterates
#

Lean declarations

  • RP.PicardIterates

Given \(T:X\to X\) and \(x_0\in X\), define the iterates \(x_{n+1}:=T(x_n)\) for \(n\ge 0\).

Lemma 0.1.5 Geometric series bound
#

Lean declarations

  • RP.GeometricSeriesBound

If \(0\le L{\lt}1\) then for all \(n\ge 1\),

\[ \sum _{k=0}^{n-1} L^k \; \le \; \frac{1}{1-L}. \]
Lemma 0.1.6 Cauchy estimate for Picard iterates
#

Uses

  • Definition 0.1.3
  • Definition 0.1.4
  • Lemma 0.1.5

Lean declarations

  • RP.PicardIsCauchy

Let \((X,d)\) be a metric space and \(T:X\to X\) be Lipschitz with constant \(L{\lt}1\). For any \(x_0\in X\) and \(m{\gt}n\ge 0\),

\[ d(x_m,x_n) \; \le \; \sum _{k=n}^{m-1} L^{\, k-n}\, d(x_{n+1},x_n) \; \le \; \frac{L^{\, n}}{1-L}\, d(x_1,x_0), \]

hence \((x_n)\) is Cauchy.

Lemma 0.1.7 Uniqueness of fixed points for contractions
#

Uses

  • Definition 0.1.3

Lean declarations

  • RP.FixedPointUnique

If \(T:X\to X\) is a contraction with constant \(L{\lt}1\), then \(T\) has at most one fixed point: if \(Tx^\ast =x^\ast \) and \(Ty^\ast =y^\ast \), then

\[ d(x^\ast ,y^\ast )=d(Tx^\ast ,Ty^\ast )\le L\, d(x^\ast ,y^\ast ) \; \Rightarrow \; x^\ast =y^\ast . \]

Contraction Mapping Theorem

Theorem 0.1.8 Contraction Mapping Theorem
#

Uses

  • Definition 0.1.2
  • Definition 0.1.3
  • Definition 0.1.4
  • Lemma 0.1.6
  • Lemma 0.1.7

Lean declarations

  • RP.ContractionMapping

Let \((X,d)\) be a complete metric space and let \(T:X\to X\) be a contraction with constant \(L{\lt}1\). Then:

  1. For any \(x_0\in X\), the Picard iterates \(x_{n+1}=T(x_n)\) converge to a limit \(x^\ast \in X\).

  2. The point \(x^\ast \) is the unique fixed point of \(T\).

  3. A priori rate: for all \(n\ge 0\),

    \[ d(x_n,x^\ast )\; \le \; \frac{L^{\, n}}{1-L}\, d(x_1,x_0). \]
Proof ▶

By Lemma 0.1.6, \((x_n)\) is Cauchy. Completeness (Definition 0.1.2) yields a limit \(x^\ast \in X\). Continuity of \(T\) (implied by Lipschitz) and \(x_{n+1}=T(x_n)\) give \(x^\ast =T(x^\ast )\), proving existence. Uniqueness follows from Lemma 0.1.7. The estimate in (iii) is the tail bound from Lemma 0.1.6.

  • RP.FixedPointRadii

Let \(r{\gt}0\). Assume \(\| T(\bar x)-\bar x\| \le Y_0\) and \(\sup _{c\in B_r(\bar x)}\| DT(c)\| \le Z(r)\). If \((Z(r)-1)r+Y_0{\lt}0\), then \(T(B_r(\bar x))\subseteq B_r(\bar x)\) and \(T\) is a contraction on \(B_r(\bar x)\) with Lipschitz constant \(\le Z(r){\lt}1\).

Lemma 0.1.19 Neumann-series invertibility
#

Lean declarations

  • RP.NeumannInvertible

If \(\| I-B\| {\lt}1\) for a matrix \(B\), then \(B\) is invertible (and \(B^{-1}=\sum _{k\ge 0}(I-B)^k\)).

Theorem 0.1.20 Radii polynomials in finite dimensions
#

Uses

  • Definition 0.1.16
  • Lemma 0.1.17
  • Lemma 0.1.18
  • Lemma 0.1.19
  • Theorem 0.1.8

Lean declarations

  • RP.RadiiPolynomialFiniteDim

If there exists \(r_0{\gt}0\) with \(p(r_0){\lt}0\), then:

  1. \(T\) maps \(B_{r_0}(\bar x)\) into itself and is a contraction there;

  2. there exists a unique \(\tilde x\in B_{r_0}(\bar x)\) with \(T(\tilde x)=\tilde x\);

  3. \(f(\tilde x)=0\), and \(\| I-ADf(\tilde x)\| {\lt}1\), hence \(Df(\tilde x)\) is invertible.

Proof ▶

From \(p(r_0){\lt}0\) we get \(Z(r_0)=Z_0+Z_2(r_0)r_0{\lt}1\) and \((Z(r_0)-1)r_0+Y_0{\lt}0\). Apply Lemma 0.1.18 with \(r=r_0\) to obtain (i) and (ii); the existence and uniqueness follow from Theorem 0.1.8. Then \(T(\tilde x)=\tilde x\) implies \(A f(\tilde x)=0\), hence \(f(\tilde x)=0\) (since \(A\) is injective); finally \(\| I-ADf(\tilde x)\| {\lt}1\) implies \(Df(\tilde x)\) is invertible by Lemma 0.1.19.
