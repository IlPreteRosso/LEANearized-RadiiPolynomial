2 Contraction Mapping Theorem

The contraction mapping theorem is a fundamental tool for proving existence and uniqueness of fixed points. This section establishes the basic definitions and the classical theorem.

Definition 2.1 Complete metric space

A metric space

\(\)(X, d)\( is \textit{complete} if every Cauchy sequence converges in \)X\(, i.e., if given any \)ε> 0\( there exists \)N(ε)\( such that \)n, m > N(ε)\( implies \)d(x_n, x_m) < ε\(, then there is some \)y X\( with \)_n→∞ x_n = y\(. \)

Definition 2.2 Contraction mapping
✓
#

Lean declarations

  • ContractingWith

Let

\(\)(X, d)\( be a metric space. A function \)T: X →X\( is a \textit{contraction} if there is a number \)κ[0, 1)\(, called a \textit{contraction constant}, such that \begin{equation*} d(T(x), T(y)) \leq \kappa \cdot d(x, y) \end{equation*} for all \)x, y X\(. \)

Theorem 2.3 Contraction Mapping Theorem
✓
#

Lean declarations

  • ContractingWith.exists_fixedPoint'

Let

\(\)(X, d)\( be a complete metric space. If \)T: X →X\( is a contraction with contraction constant \)κ\(, then there exists a unique fixed point \)x X\( of \)T\(. Furthermore, \)x\( is globally attracting, and for any \)x X\(, \begin{equation*} d(T^n(x), \tilde{x}) \leq \frac{\kappa ^n}{1-\kappa } d(T(x), x). \end{equation*} \)

Remark 2.4
#

Given a contraction mapping

\(\)T: X →X\(, the rate at which points in \)X\( converge to the globally attracting fixed point \)x\( is determined by \)κ\(. In particular, the smaller \)κ\( is, the faster iterates under \)T\( converge to \)x.
